##iPhone Purchase Prediction Using Logistic Regression
## Project Overview

This beginner machine learning project predicts whether a customer will purchase an iPhone based on three features:
* Gender
* Age
* Salary

The project uses logistic regression for binary classification.

## Dataset
The dataset contains 400 customer records and the following columns:
* Customer gender
* Customer age
* Estimated annual salary
* Purchase decision: 0 or 1 

The target variable is Purchase iphone:
* 0: The customer will not purchase an iPhone
* 1: The customer will purchase an iPhone

## Model Evaluation
The model is evaluated using:
* Confusion matrix
* Accuracy
* Precision
* Recall
* F1-score
In the current experiment, the model achieved approximately 91% accuracy on the test data.

## Libraries Used
* Python
* Pandas
* NumPy
* Scikit-learn
* GoogleColab

## Important Note
This project was created for educational purposes. It should not be used for real-world customer decisions without further validation and fairness analysis.
